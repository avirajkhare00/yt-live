## yt-live

### Overview

It's a small full stack prototype built on top of express framework.

### Installation

 - `git clone https://github.com/avirajkhare00/yt-live`
 - `docker-compose up`

### Usage

Just run the command: `docker-compose up`

Relax and Enjoy
